# Passweird

Password Manager built with Next.js and shadcn/ui.

## Tech Stack

- Next.JS with server action
- Prisma ORM
- Tailwind CSS
- shadcn/ui
- neon postgres

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
